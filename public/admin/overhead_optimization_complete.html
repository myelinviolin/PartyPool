How It Works Now
  1. Starts Conservative: Calculates vehicles based on 3-4 participants per driver
  2. Runs Optimization: Assigns passengers using geographic clustering
  3. Checks Overhead: Measures each driver's extra time
  4. Auto-Adjusts: If anyone has >20 min overhead, adds vehicles
  5. Re-optimizes: Continues until all overhead <20 min
This happens automatically when you click "Run Optimization"!
