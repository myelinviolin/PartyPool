Manual Trigger Test for Button State

Instructions:

  1. Open the Admin Dashboard in another tab
  2. Open browser console (F12)
  3. Try these commands in the console:
Console Commands to Test
1. Check if functions exist:
typeof window.checkFormChanged
typeof window.storeOriginalValues
typeof window.updateEvent
2. Check original values:
originalEventData
3. Manually trigger change detection:
// Change a field value manually
document.getElementById('eventName').value = 'New Test Name';

// Then call the check function
checkFormChanged();
4. Force enable button:
const btn = document.querySelector('button[onclick*="updateEvent"]');
btn.disabled = false;
btn.classList.remove('btn-secondary');
btn.classList.add('btn-primary');
5. Debug current state:
console.log('Original:', originalEventData);
console.log('Current Name:', document.getElementById('eventName').value);
console.log('Current Address:', document.getElementById('eventAddress').value);

// Check if values are different
const current = {
    event_name: document.getElementById('eventName').value,
    event_address: document.getElementById('eventAddress').value,
    event_date: document.getElementById('eventDate').value,
    event_time: document.getElementById('eventTime').value
};
console.log('Current:', current);
console.log('Are they different?', JSON.stringify(originalEventData) !== JSON.stringify(current));
