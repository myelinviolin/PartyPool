üîß Database Operations

The removal process handles multiple database tables in a transaction:

  1. carpool_passengers: Remove as passenger from any assignments
  2. carpool_assignments: Remove any assignments where they're the driver
  3. users: Delete the user record
  4. optimization_results: Clear outdated optimization
  5. events: Set optimization_status back to 'pending'

All operations are wrapped in a transaction for data integrity.
