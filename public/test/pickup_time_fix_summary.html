Problem Identified:

The optimizeForExactTarget() method was calculating overhead time based only on distance, not including the 3-minute stops for passenger pickups. This meant drivers appeared to have no time overhead when picking up passengers.

Solution Implemented:

Updated the overhead calculation to include:

  • 3 minutes per passenger pickup - Added to total travel time
  • Proper overhead calculation - overhead_time = total_travel_time - direct_time
  • Individual pickup times - Each passenger gets a calculated pickup time
  • Updated departure times - Adjusted based on total travel time including pickups

Code Changes:

// Calculate travel time with 3-minute pickup stops $travel_time = $this->calculateTravelTime($total_distance); $num_passengers = count($assignments[$i]['passengers']); $pickup_time = $num_passengers * 3; // 3 minutes per passenger pickup $total_travel_time = $travel_time + $pickup_time; // Update travel times $direct_time = $this->calculateTravelTime($assignments[$i]['direct_distance']); $overhead_time = $total_travel_time - $direct_time; $assignments[$i]['overhead_time'] = round($overhead_time) . ' minutes';
