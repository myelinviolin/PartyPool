What is the 2-Opt Algorithm?

The 2-opt algorithm is a local search algorithm that improves a route by iteratively swapping pairs of edges. It examines all possible ways to reconnect two edges in the tour and chooses the swap that produces the shortest total distance.

How It Works:

  1. Start with initial pickup order (as assigned by clustering)
  2. Try all possible edge swaps between passenger pickups
  3. Keep swaps that reduce total distance
  4. Repeat until no improvement is possible
