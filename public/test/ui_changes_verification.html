Summary of Changes

Two UI improvements have been successfully implemented:

  1. Route Selector Panel: Fixed hover behavior so only individual route items highlight when directly hovered over, preventing the entire panel from appearing to be interactive.
  2. Carpool Assignments: Removed the vehicle capacity badges from the home page assignment cards for a cleaner, simpler display.

Both changes improve the user experience by reducing visual clutter and providing more predictable interaction behavior.
